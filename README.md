# dotfiles
arch + hyprland setup

## setup

### 0. Dependencies

Ensure that both stow & git are installed as well as the necessary packages for the specific dotfiles.

```bash
sudo pacman -S stow git kitty zsh
```

### 1. Clone the repository
```bash
git clone 
```

### 2. Install the dotfiles
```bash
cd dotfiles
stow .
```
